# Student Management System

This is a web-based Student Management System built using JavaScript and object-oriented programming principles.

## Features
- Add, update, delete students
- Search and filter students
- Dashboard with analytics (GPA, departments)
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript (OOP)

## How to Run
1. Download the project
2. Open index.html in your browser

## Purpose
This project demonstrates CRUD operations, OOP concepts, and UI design for managing student data.
